# AmigoSecreto
Aplicación web para sortear un amigo secreto entre varios participantes.


### Este proyecto usa:
- HTML.
- CSS.
- JavaScript.


### ¿Comó Usarlo?
- Ingresa al sitio web.
- Ingresa los nombres de los participantes en el input.<img width="1918" height="1023" alt="image" src="https://github.com/user-attachments/assets/7a0ff070-ff2b-4283-9d84-1ed47f3c6c64" />

- Has clic en "Agregar Amigo". <img width="1856" height="977" alt="image" src="https://github.com/user-attachments/assets/5254cdd4-3264-487d-8955-161d15b05361" />

- Una vez ya hayas agregado a todos los participantes, has clic en "Sortear Amigo". <img width="1845" height="994" alt="image" src="https://github.com/user-attachments/assets/ea189c08-a881-4b80-814e-5f57fa64fc80" />

- El programa mostrará un amigo secreto aleatorio sin repetir hasta agotar la lista. <img width="1841" height="991" alt="image" src="https://github.com/user-attachments/assets/28303aa1-c38f-41de-b6bd-9b8fe2dc86b4" />

- Cuando el programa te diga "Ya han sido seleccionados todos los nombres, por favor recargue la pagina para volver a empezar", simplemente debes presionar F5 y ya puedes volver a usar el programa.<img width="1837" height="999" alt="image" src="https://github.com/user-attachments/assets/aebe9436-ed6e-4db3-83b2-8062461cfe11" />
